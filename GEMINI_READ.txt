## ✅ A. Core Engine Target’ları

### 1. `axiom_engine_core`
Engine’in saf hesaplama / runtime çekirdeği.

İçeriği:
- scheduler
- dispatch
- runtime
- memory arenas
- command execution
- state model

**Bağımlılıkları minimum tut.**
Qt bağımlılığı YOK.

```cmake
add_library(axiom_engine_core STATIC
    engine/core/*.cpp
    engine/dispatch/*.cpp
    engine/runtime/*.cpp
    engine/memory/*.cpp
)

target_include_directories(axiom_engine_core PUBLIC
    ${CMAKE_SOURCE_DIR}/engine/include
)

target_link_libraries(axiom_engine_core PUBLIC
    mimalloc
    robin_map
)
```

---

### 2. `axiom_engine_compute`
Numerical / HPC / SIMD / Eigen / asmjit / advanced compute burada.

```cmake
add_library(axiom_engine_compute STATIC
    engine/compute/*.cpp
    engine/math/*.cpp
    engine/simd/*.cpp
)

target_link_libraries(axiom_engine_compute PUBLIC
    axiom_engine_core
    Eigen3::Eigen
    asmjit
)
```

---

### 3. `axiom_engine_ipc`
Zero-copy IPC ve shared memory burada.

```cmake

    engine/ipc/*.cpp
    engine/ringbuffer/*.cpp
    engine/shared_memory/*.cpp
)

target_link_libraries(axiom_engine_ipc PUBLIC
    axiom_engine_core
)
```

Burada Qt YOK.  
Hot path burada yaşar.

### 4. `axiom_engine_api`

## ✅ Ne Nereye Taşınmalı?

### Şu anki `core/dispatch`
→ `engine/dispatch/`

### Şu anki `shaders`
→ ikiye ayrılmalı:
- engine compute shader ise `engine/shaders/`
- expressway render shader ise `expressway/shaders/`

### Şu anki `studio/axiom_studio/backend`
→ `products/studio/backend`

### Şu anki `studio/axiom_studio/frontend`
→ `products/studio/frontend`

### Şu anki `src/gui`
→ ya `products/studio/src` ya da tamamen kaldırılmalı  
(tek GUI entrypoint seç)

### Şu anki `gui/qt`, `gui/python`
→ bunlar legacy/tooling ise `tools/gui/` gibi yere taşınmalı  
ürün runtime ile karışmamalı.

# 3) Önerilen Root CMake Yapısı

Kök CMake aşağı yukarı şöyle olmalı:

```cmake
cmake_minimum_required(VERSION 3.20)
project(Axiom LANGUAGES C CXX)

option(AXIOM_BUILD_TESTS "Build tests" ON)
option(AXIOM_BUILD_STUDIO "Build Studio" ON)
option(AXIOM_BUILD_DASHBOARD "Build Dashboard" ON)
option(AXIOM_BUILD_LENS "Build Lens" ON)

add_subdirectory(axui)
add_subdirectory(engine)
add_subdirectory(expressway)

if(AXIOM_BUILD_DASHBOARD)
    add_subdirectory(products/dashboard)
endif()

if(AXIOM_BUILD_STUDIO)
    add_subdirectory(products/studio)
endif()

if(AXIOM_BUILD_LENS)
    add_subdirectory(products/lens)
endif()

if(AXIOM_BUILD_TESTS)
    add_subdirectory(tests)
endif()
```
