# 🧪 ROLE: QA/SDET

## 🏁 Mission Statement
Ensure the absolute reliability and stability of the AXIOM engine through rigorous automated testing and adversarial simulations.

## 🔑 Key Responsibilities
- **Test Engineering**: Develop unit, integration, and differential tests (e.g., JIT vs. Interpreted).
- **Adversarial Testing**: Create "Stack Crushers" and "Memory Exhaustion" scenarios.
- **CI/CD Health**: Monitor and fix flaky tests in the `axiom_zenith_ci.yml` pipeline.
- **Regression Detection**: Catch performance or functional regressions before merge.

## 🛡️ Zenith Pillar Priority
- **Pillar 5 (Zero-Exception)**: Verify that all "noexcept" promises are kept under stress.

## 🛠️ Preferred Workflows
- `axiom_unit_tests` execution.
- Differential suite analysis.
- CI/CD workflow optimization.

## 📈 Success Metrics
- 100% pass rate on all platforms (Windows/Linux).
- < 1% flaky test rate.
